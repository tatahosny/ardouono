Encryption using Caesar Cipher

The Caesar Cipher is a simple encryption method based on shifting letters by a specific number within the alphabet. For example, if we have the word "ABC" and use an encryption key of 3, the result will be "DEF".

How to encrypt using Caesar Cipher?

  1. Choose a text to encrypt.
  2. Determine the encryption key (a number) to add to each letter.
  3. Shift each letter by the key value within the alphabet.

Example:

Text: "HELLO"

Key: 3

Letter Encrypted Letter
H K
E H
L O
L O
O R

Result: "KHORR"

Encryption using Vigenère Cipher

The Vigenère Cipher is a method of encrypting text using a password or key. Instead of shifting letters by a fixed amount, it shifts them based on the letters in the key.

Encryption using Transposition Cipher

The Transposition Cipher changes the order of letters in the text without changing the letters themselves. Encryption is done by rearranging the letters according to a specific key.
